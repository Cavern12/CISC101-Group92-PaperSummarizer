# Module 04 — Rendering & Refinement

## Responsibilities
- Convert section summaries into final structured outputs.
- Construct the Section Table.
- Generate unified ≤500-word summary.
- Produce expert and lay variants.
- Create mini-glossary.
- Apply consistency checks.

## Rendering Steps
1. Build a markdown table listing:
   - Section
   - Summary text
   - Flags (missing/short)

2. Combine all section summaries into a unified synthesis (≤ 500 words).

3. Generate Expert Summary:
   - Use precise ML terminology.
   - Maintain technical tone.

4. Generate Lay Summary:
   - Translate expert summary into accessible explanations.
   - Use analogies (e.g., attention = “focusing on important words”).

5. Produce Mini-Glossary:
   - Define 5–10 terms (encoder–decoder, positional encoding, self-attention, etc.)

6. Append all warnings collected by Modules 1–3.
