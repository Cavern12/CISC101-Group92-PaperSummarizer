# CISC 101 – Research Paper Summarizer (Group 92)

This repository contains the full modular Research Paper Summarizer system developed using
specification design, TDD, modular prompting, and multi-module architectures (PS2 → PS7).

## Features
- Multi-section summarization (≤150 words per section)
- Unified summary (≤500 words)
- Summary levels: "short" or "detailed"
- Expert + Lay summaries
- Strict evidence mode (no hallucination)
- Missing/short section warnings
- Citation extraction module
- Key contributions summarizer
- Travel Planner–style modular architecture
